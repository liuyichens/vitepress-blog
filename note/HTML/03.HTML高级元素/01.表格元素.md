#

## 表格元素中的常见元素

table, tr(table row), td(table data)

## 表格中重要的属性

`border-collapse` CSS 属性用来决定表格的边框是分开还是合并的<br/>
`table{border-collapse: collapse}` 用于合并单元格的边框

## 表格的其他元素

- thead 表格头部
- tbody 表格主体
- tfoot 表格页脚
- caption 表格的标题
- th 表格的表头单元格
