#
## CSS 颜色表示

> CSS 中颜色的表示方法

- 颜色关键字
- RGB 颜色

### RGB 表示法

- 十六进制符号 `#RRGGBB[AA]` `#ff0000` 等价于 `#ff0000ff`
- 十六进制符号 `#RGB[A]` 这种方式是上面一种方式的简写
- 函数符: `rgb[a](R, G, B[,A])` 255 表示 100% a 的值的范围是 0-1 或者百分比， 1 相当于 100% 完全不透明
